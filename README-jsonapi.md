# JSON API

## Requirements

For the MariaDB/MySQL backend:
```
# apt-get install mariadb-{server,client}
```

~~~For the MongoDB backend (stale): ~~~

~~~apt-get install mongodb ~~~

NodeJS:

```bash
# apt-get install nodejs npm
```

External Node modules:

```
$ cd json-api
$ npm install
```



## Running the API

```
$ cd json-api
$ nodejs server.js
```
